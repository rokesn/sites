[⚠️] ACTION 1 — Get Naver verification code
     1. Go to: https://searchadvisor.naver.com/
     2. Click "사이트 등록" (Register site)
     3. Enter: https://밤포탈.여기여.site/
     4. Copy the verification code
     5. In index.html replace:
        YOUR_NAVER_VERIFICATION_CODE → your real code

[⚠️] ACTION 2 — Get Google Search Console code
     1. Go to: https://search.google.com/search-console/
     2. Click "Add property"
     3. Enter: https://밤포탈.여기여.site/
     4. Choose "HTML tag" verification method
     5. Copy the content value
     6. In index.html replace:
        YOUR_GOOGLE_SEARCH_CONSOLE_CODE → your real code

[⚠️] ACTION 3 — Get Daum/Kakao verification code
     1. Go to: https://webmaster.daum.net/
     2. Register your site
     3. In index.html replace:
        YOUR_DAUM_VERIFICATION_CODE → your real code

[⚠️] ACTION 4 — Submit sitemap to all search engines
     After deploying the site, submit sitemap at:
     • Google:  https://search.google.com/search-console/ → Sitemaps
                Submit: https://밤포탈.여기여.site/sitemap.xml
     • Naver:   https://searchadvisor.naver.com/ → 사이트 등록 → 사이트맵
     • Bing:    https://www.bing.com/webmasters/ → Sitemaps
     • Daum:    https://webmaster.daum.net/

[⚠️] ACTION 5 — Convert SVG images to JPG/PNG for better compatibility
     og-image.svg → og-image.jpg (1200x630)
     logo.svg → logo.png (600x60)
     apple-touch-icon.svg → apple-touch-icon.png (180x180)
     Use: https://cloudconvert.com/svg-to-jpg
     Or install sharp: npm install sharp
     Then update references in index.html back to .jpg/.png

[⚠️] ACTION 6 — Add Google Analytics 4 (GA4)
     1. Go to: https://analytics.google.com/
     2. Create a new property for 밤포탈.여기여.site
     3. Get your Measurement ID (G-XXXXXXXXXX)
     4. Add before </head> in index.html:
        <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
        <script>
          window.dataLayer = window.dataLayer || [];
          function gtag(){dataLayer.push(arguments);}
          gtag('js', new Date());
          gtag('config', 'G-XXXXXXXXXX');
        </script>

[⚠️] ACTION 7 — Minify CSS and JS before deployment
     • CSS: https://cssminifier.com → paste styles.css → save as styles.min.css
     • JS:  https://jsminifier.com  → paste script.js  → save as script.min.js
     • Update index.html: styles.css → styles.min.css, script.js → script.min.js

[⚠️] ACTION 8 — Update dateModified in WebPage schema when content changes
     In index.html, find:
       "dateModified": "2024-01-01"
     Update to today's date every time you change content.
     Also update the 최신 업데이트 section date.

[⚠️] ACTION 9 — Add Content Security Policy (CSP) header on server
     Ask your hosting provider to add this HTTP header:
     Content-Security-Policy: default-src 'self'; script-src 'self' 'unsafe-inline'
       https://www.googletagmanager.com https://static.cloudflareinsights.com;
       style-src 'self' 'unsafe-inline' https://fonts.googleapis.com
       https://cdnjs.cloudflare.com; font-src https://fonts.gstatic.com
       https://cdnjs.cloudflare.com; img-src 'self' data: https:;

[⚠️] ACTION 10 — Create Naver Blog for backlinks
     Naver Blog (네이버 블로그) posts rank very high in Naver search.
     Create a blog at: https://blog.naver.com/
     Post about 밤포탈 with links back to your site.
     This is the #1 off-page SEO strategy for Korean search.
